# rust-yt-dlp Video Downloader

rust-yt-dlp is a feature-rich video downloader that supports thousands of sites and supports graphical operations. The project is a new rust language downloader based on modifications and improvements to yt-dlp and ffmpeg.

## Function
- Supports videos and playlists (currently only single video)
-Support graphics operations
- Support customization of installation location
- Support custom output path
- Multiple image quality options (Best 4K / 2K / 1080P / 720p / 480p, etc.)
- Automatically download the complete video after merging yt-dlp video and ffmppe audio, and solve all pain points with one installation.
- Cross-platform support: Windows, Linux, macOS
