# rust-yt-dlp Video Downloader

一个用 Rust + egui 开发的简单 YouTube（及支持站点）视频下载工具。

## 功能
- 支持视频、直播、播放列表（当前仅单视频）
- 多种画质选择（Best / 720p / 480p 等）
- 自动下载 yt-dlp
- 跨平台支持：Windows、Linux、macOS

- Linux / macOS：给执行权限 `chmod +x rust-yt-dlp-*`，然后 `./rust-yt-dlp-*` 运行。
